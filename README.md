# AR Learning Application

This is a Web-Based Augmented Reality Application using AR.js and A-Frame.

## Features
- Multiple image tracking markers
- 3D models visualization
- Interactive UI panel
- Mobile-friendly AR experience

## How to Use
1. Open the project link
2. Allow camera access
3. Scan markers to see AR models

## Technologies Used
- HTML, CSS, JavaScrip
- A-Frame
- AR.js

# deployed site link
https://ar-learning-app.onrender.com/

To use it, simple open the site and then show the camera to the marker(1,2,3).png in the assets folder
